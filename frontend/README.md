# AgriChain 🌾

AgriChain is a blockchain-enabled farm-to-consumer traceability platform
designed to ensure transparency, trust, and tamper-proof food supply chains.

## Problem
Farmers lack proof of origin.
Consumers lack trust.
Data can be manipulated centrally.

## Solution
AgriChain uses:
- Ethereum blockchain for immutable batch verification
- MongoDB-backed backend for operational data
- QR-based public trace access
- Role-based dashboards (Farmer, Middleman, Admin)

## Architecture
Frontend → Backend API → MongoDB  
Blockchain layer used for verification & audit trail

## Tech Stack
- React + Vite + Tailwind CSS
- Node.js + Express + MongoDB
- Ethereum (Solidity, Smart Contracts)
